# Multimodal Harmful Content Detection Pipeline
## SW중심대학 산학 R&D 프로젝트 – Research Archive

<img src="무하유_유해콘텐츠_데이터_모델선정/최종_모델/project_banner.jpg" width="600" alt="Project Banner">

본 저장소는 **2025년도 SW중심대학 산학 R&D 프로젝트**의 연구 결과를
아카이브 형태로 정리한 레포지토리입니다.

이미지 및 비디오 기반 멀티모달 유해 콘텐츠 자동 탐지 시스템 (Hazard Killer)의 기획, 데이터 구성, 모델 비교 실험,
그리고 최종 시스템 설계를 연구 기록 관점에서 정리합니다.

> ⚠️ 본 저장소는 **종료된 연구 프로젝트의 아카이브**이며,  
> 실행 중심 레포지토리가 아닌 **연구 설계·실험·성과 정리**를 목적으로 합니다.

---

## Project Overview

온라인 플랫폼에서 증가하는 폭력, 무기, 혈액/상해, 약물, 성적 콘텐츠,
위협 및 위험 행동과 같은 유해 콘텐츠를 자동으로 탐지하기 위해 **모달별 특성을 반영한 멀티모달 데이터 파이프라인**을 설계·구현하였습니다.

본 프로젝트는 **산학 협력(무하유)** 기반 캡스톤 과제로 수행되었으며,
실서비스 적용 가능성을 고려하여 다음 원칙을 중심으로 연구를 진행하였습니다.

- 이미지 / 비디오 모달 분리 설계
- 벤치마크 의존을 최소화한 **독립 평가 데이터셋 구성**
- 규칙 기반 추론과 딥러닝 모델의 결합
- 실제 서비스 안정성을 고려한 임계값(threshold) 설계

---

## System Architecture Summary

### Image Pipeline
- YOLOv8 기반 유해 객체 탐지
- CLIP 기반 장면 및 문맥 이해
- Zero-shot 프롬프트 기반 행동 추론
- 특징 결합 후 MLP 분류기 적용

### Video Pipeline
- 32-frame 균등 샘플링 기반 입력 표준화
- CLIP / ViT / SlowFast 기반 late fusion
- p95 통계 기반 점수 집계
- 실서비스 안정성을 고려한 threshold 설계

---

## Dataset & Evaluation

### Training Data
- Public datasets: HOD, COCO Safe, RWF-2000, RLVS

### Evaluation Data
- 팀원이 직접 수집 및 라벨링한 **독립 평가 데이터셋**
- 총 **1,200 samples (3명 × 400개)**
- 학습 데이터와 완전히 분리하여 사용

### Performance Summary
- Image model F1-score: **0.6912**
- Video model F1-score: **0.5813**

> 독립 평가 데이터는 **모델 선정 및 성능 비교 목적**으로만 사용되었으며,
> 학습 과정에는 포함되지 않았습니다.
